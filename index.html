Runa Antennae

Summary

This package is an antennae addon for the Runa model by Torinyan. You must own the Runa base model to use it.

Installation

  1. Unpack the dependencies, in order.
  2. Unpack Runa_Antennae.unitypackage.
  3. Drag Assets/Kemono_Io/Runa_Antennae.prefab into the Runa_Source object.
  4. Constrain the Runa_Antennae object to the head.
    1. Select the Runa_Antennae object in the Hierarchy panel.
    2. Drag the hips bone into the Parent Constraint's Source in the Inspector panel.
    3. Click the Activate button.
  5. Set the Runa_Antennae's anchor override.
    1. Select the Runa_Antennae mesh in the Hierarchy panel.
    2. Drag the hips bone into the Anchor Override in the Inspector panel.
  6. Add the antennae to the emission and hue shift animations.
    1. Select the Runa_Source object in the Hierarchy panel.
    2. Set the Animator's Controller to FX_Runa in the Inspector panel.
    3. Select the Animation tab and click the lock icon if it is not already locked.
    4. Select the Runa_Antennae object in the Hierarchy panel.
    5. Select the color_0 animation and click the record button.
    6. Set the Hue Shift properties to 0 on both materials.
    7. Select the color_1 animation and click the record button.
    8. Set the Hue Shift properties to 1 on both materials.
    9. Select the body_lightning_0 animation and click the record button.
    10. Set the Emission Strength property on the Runa_Antennae material to 0.
    11. Set the Emission Strength property on the Runa_Antennae_Orb material to 0.
    12. Select the body_lightning_1 animation and click the record button.
    13. Set the Emission Strength property on the Runa_Antennae material to 0.25.
    14. Set the Emission Strength property on the Runa_Antennae_Orb material to 4.
    15. Click the Preview button to exit record mode.
    16. Click the lock icon.
    17. Right click the Animator's Controller and select Revert.
